# Security

## Reporting a vulnerability

Please email **contact@bartrosa.dev** with:

- A short description of the issue
- Steps to reproduce (if possible)
- Affected component (e.g. CI workflow, dependency)

This is a small open-source project; maintainers will respond on a **best-effort** basis.

## Scope

- **In scope:** issues in this repository (code, workflows, documented usage).
- **Out of scope:** third-party services (LLM providers, GitHub itself) — report those to the respective vendor.

## Supported versions

Only the latest `main` branch is actively maintained during early development (`0.x`).
