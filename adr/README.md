# Architecture Decision Records

This directory records significant architectural decisions.

New ADRs use this shape:

```markdown
# ADR-XXXX — Title

## Status

Proposed | Accepted | Deprecated | Superseded

## Context

What forces this decision?

## Decision

What we chose.

## Consequences

Trade-offs and follow-ups.
```
